# 專案客製化設定與對話紀錄 (Project Rules)

## 專案概述
本專案為「拓元餘票查詢器」的 Chrome 擴充功能，目前專門為「五月天[回到那一天]｜台北站」客製化追蹤與顯示。

## 介面與排版規則 (UI & Layout Guidelines)
未來的編輯與修改，請遵守以下從過去對話中確立的規則：

1. **擴充功能視窗尺寸**：當作為擴充功能 (`isExtension === true`) 執行時，畫面的主要寬度固定為 `w-[700px]`，並且隱藏底部水平卷軸 (`overflow-x-hidden`)。
2. **卡片網格排列 (Grid Layout)**：
   - 擴充功能內的卡片統一強制為「一列 3 個小卡」 (`grid-cols-3`) 往下排列。
   - 卡片網格的左右邊緣 (margin/padding) 必須與上方標題白底區塊的左右邊緣「切齊對齊」，總寬均限制在 `700px` 以內。
3. **字體與大小 (Typography)**：
   - 卡片內的字體避免過度縮小。日期與標題維持在 `13px`；數字以 `text-base` 與 `text-lg` 為主，確保閱讀舒適度。
4. **特定文字與文案**：
   - 標題文字固定為：「五月天[回到那一天]｜台北站」。
   - 下載按鈕的縮小版文字為：「下載各區餘票 (CSV)」。
5. **移除特定功能**：
   - 原先的齒輪（設定）功能與手動設定彈出選單已移除，擴充功能預設即為最大預設呈現。

> **備註給 AI 助手**：在進行接下來的任何變更時，務必將這些 UI/UX 的屬性一併考量進去，不得破壞此 700px 寬度的「列 3 卡」與對齊限制。
